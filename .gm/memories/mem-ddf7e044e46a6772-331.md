Any git history rewrite or force-push sourced from an external clone or mirror verifies that source tip is a real ancestor of live HEAD via git merge-base is-ancestor before filter-repo or force-push runs against it. A freshly named clone directory is not proof the clone is fresh, staleness is a content property not a naming one.
