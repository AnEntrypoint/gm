SKILL.md auto-refresh mechanism (gm-plugkit bootstrap): every bootstrapPlugkit call compares the sha256 of the bundled gm-skill/skills/gm-skill/SKILL.md (shipped inside the npm package) against the installed copies at ~/.agents/skills/gm-skill/SKILL.md and ~/.claude/skills/gm-skill/SKILL.md. Hash mismatch triggers an atomic write (.tmp + rename) of both targets so the agent sees the latest prose next session - no manual reinstall. Logged to bootstrap.jsonl as SKILL.md refreshed. The bundled SKILL.md is source of truth; reinstalling gm-skill only matters when the npm package itself changes, which the cascade pipeline guarantees on every plugkit version bump.
