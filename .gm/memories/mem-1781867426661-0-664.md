LEGACY RETIREMENT (2026-06-19): the 15 legacy downstream gm-skill variants (gm-cc, gm-gc, gm-oc, gm-codex, gm-kilo, gm-qwen, gm-hermes, gm-thebird, gm-vscode, gm-cursor, gm-zed, gm-jetbrains, gm-copilot-cli, gm-antigravity, gm-windsurf) are retired and their repos archived. gm-skill (AnEntrypoint/gm) is the single canonical package. Users install via `bun x skills add AnEntrypoint/gm` into any harness. AnEntrypoint/gm-skill is a back-compat mirror receiving only SKILL.md per release. Three npm packages publish from the gm repo root: gm-skill (harness), gm-plugkit (bootstrap + watcher), plugkit-wasm (wasm binary) -- all shipped on every version-bump commit.
