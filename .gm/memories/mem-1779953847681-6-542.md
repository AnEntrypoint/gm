gm skill rule: to debug client-side code, expose state as window.* globals and read it live via the browser verb page.evaluate, running experiments in the browser, rather than blind experimentation plus continuous server restarts. A global plus one page.evaluate reads actual runtime state in a single dispatch where a restart-and-eyeball loop burns a turn and observes nothing. The live page is the debugger; the same browser surface that witnesses an edit also diagnoses it. Shipped in SKILL.md, AGENTS.md, and docs/paper.html section 23.1.
