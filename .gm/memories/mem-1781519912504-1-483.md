memorize-fire ingestion classifier rejects any memo framed as history rather than as a present-tense rule about what must be true now; the reply names the trigger and the memo is not stored, so rephrase into a present-tense invariant and re-fire. Dual-writing an AGENTS.md or CLAUDE.md rule into the store is subject to this gate. Never set namespace=AGENTS.md on a memorize-fire; load-bearing rules target the default namespace so recall and auto_recall surface them on later turns.
